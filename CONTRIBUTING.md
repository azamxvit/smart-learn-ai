# Contributing to SmartLearnAI

## Commit Convention
- feat(SLA-N): add new feature
- fix(SLA-N): fix a bug
- chore(SLA-N): config, tooling
- test(SLA-N): add or update tests
- docs(SLA-N): documentation

## Branch Naming
- feature/SLA-N-short-description
- fix/SLA-N-short-description

## Pull Requests
- One feature per PR
- Tests required
- Pass lint before pushing
